Сборка приложения из исходного кода осуществлялась на нижеследующих настройках:
- способ сборки: IDE Visual Studio 2017
- целевая платформа: x64
- конфигурация: Release
Сборка и запуск приложения выполнялись на ОС Windows 10.
Для этого были выполнены предварительные операции по загрузке и настройке дополнительных библиотек,
таких как OpenCV.
- с помощью CMake была собрана из исходников OpenCV
- установлена в корректное состояние переменная среды OPENCV_DIR (каталог расположения библиотеки)
- в свойствах проекта ImageViewer прописаны пути к включаемым файлам OpenCV, 
к каталогу расположения статической версии библиотеки (.lib)
